## ğŸ› ï¸ Customization

To customize this card for another person, edit the following in `index.html`:

1. **Personal Details** (Lines 82-88): Name, title, and court badges
2. **Contact Links** (Lines 92-118): Phone, email, WhatsApp numbers
3. **Practice Areas** (Lines 132-138): Legal specializations
4. **Office Address** (Lines 151-154): Chamber/office location
5. **Contact Object** (Lines 201-209): JavaScript contact data for vCard

Replace `profile.jpg` with your own photo (recommended size: 400x400px).
